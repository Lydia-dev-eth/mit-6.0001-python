# MIT 6.0001 Problem Sets

This repository contains my solutions to the MIT 6.0001 course:  
Introduction to Computer Science and Programming in Python

---

## Repository Structure

- psetX/ – Problem sets (major assignments)  
- lecture_exercises/ – Exercises from lectures  
- finger_exercises/ – Finger exercises (small practice tasks)

Each PSet or exercise folder contains:
- README.md → Description and instructions  
- code/ → Python code files  
- notes/ → Personal reflections, learning notes, and debugging explanations

---

## Learning Goals

- Understand Python programming basics  
- Solve problems using loops, conditionals, and data structures  
- Build small interactive programs  
- Practice debugging using test cases  
- Track progress across different types of exercises
